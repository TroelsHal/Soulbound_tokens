# Soulbound Tokens

## Requirements

To build and run the program you will need:
+ next
+ bootstrap

You can install these dependencies with:
```bash
npm install next
npm install bootstrap --legacy-peer-deps
```

## Run
Run the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000)
